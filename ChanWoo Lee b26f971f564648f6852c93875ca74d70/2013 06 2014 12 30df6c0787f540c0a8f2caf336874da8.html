경기대학교 신문방송사 웹진운영국 거북이 (2013.06~2014.12)

소속 및 직책

  • 기술팀 소속
    • 기술 부원

담당 업무

  • 웹진 서버 운영
    • Ubuntu, APM, XpressEngine으로 구성된 웹진 운영
      • 경기대 웹진에서 취재한 뉴스 기재하는 사이트 유지보수
      • 회의를 통해 필요한 기능이나 학생의 질문과 요구사항을 반영하는 업무

사용 기술 및 도구

  • HTML, CSS, Java, SQL
  • Ubuntu
  • APM
  • XpressEngine
  • putty

업무 내용 및 사유

  • DB 칼럼 오류 수정
    • 사유: 전 기수가 XpressEngine을 업데이트하면서 DB쪽으로 필수 칼럼을 전송 안해서 게시글이 작성이 안되는 문제 발생
  • 웹진 내부 인원을 위한 인트라 게시판 생성

    사유 :

    1. 전 기수들의 개발 기록 작성 미흡으로 인해 발생한 DB칼럼 누락 문제 같은 이슈를 재발 방지
    1. 웹진 내부 정기 회의록 및 교육 내용 기록
  • 해외 IP 차단 : GeoIP API 추가
    • 사유 : 종종 러시아 쪽에서 오는 봇이 광고 메세지를 게시하는 문제가 발생.
  • 레이아웃 재구축
    • 사유 : 내부 회의 결과 사이드 바에 편의 기능 및 추후에 위젯 기능을 넣자는 요구사항 발생
  • 자동 가입 방지 태그 추가 : CAPTCHA 애드온 활성화
    • 사유 : 위에 해외의 봇의 광고 봇 광고 방지의 연장선
  • 배너 관리 모듈 추가
    • 사유 : 웹진이나 학교에서 하는 이벤트를 홍보를 위해 배너 광고를 만들어야 한다는 내부 회의의 결과
  • Footer와 Navbar 수정
    • 사유 : 웹진의 접근성을 높히고, 내가 웹진을 자주오는 만큼 학교 홈페이지에 링크가 필요하다는 의견으로 Footer에 추가, 및 Navbar의 글자 겹침 현상 때문에 글꼴과 폰트 사이즈 수정
  • 회원 아이디 레이어 수정
    • 사유 : 등급별 회원 아이콘을 등록함에 따라, 회원 아이디의 출력이 레이어를 초과해서 출력되는 현상 발생
  • 메일 서버 기능 추가
    • 사유 : 회원 가입 인증을 메일 인증으로 하자는 내부 회의에서의 의견 발안
  • 드래그 및 오른쪽 클릭 방지 기능 추가
    • 사유 : 웹진에 올라오는 정보를 다른 교내 커뮤니티에 무단으로 올리는 경우 발생해서 방지를 위해 기능 추가
    • <body oncontextmenu="return false" ondragstart="return false" onselectstart="return false">
  • 신입 기술팀 교육 및 인수인계
    • 사유 : 신입 기술팀 인원이 입부함에 따라 교육을 실시
  • 웹진 홈페이지 리뉴얼, XE레이아웃 구매 및 적용
    • 사유 :
      1. 지금까지의 웹진 홈페이지가 반응형 웹페이지가 아니라서 이용이 불편하다는 웹진 홈페이지 이용 학생들의 클레임 발생
      1. 편집팀의 신규 기사를 홈페이지 메인에 크게 강조해서 보여 줄 수 있는게 좋을 것 같다는 요구사항이 내부 회의에서 발안
  • 썸네일 기능 추가
    • 사유 : 편집팀이 기사를 업로드 하는 과정에 메인으로 출력되는 이미지를 정하고 싶다는 요구사항이 내부 회의에서 발안
  • 팝업 광고 기능 추가
    • 사유 : 학교 축제가 다가옴에 따라 확교 홈페이지에 대한 홍보가 필요하다는 의견이 내부 회의에서 발안
