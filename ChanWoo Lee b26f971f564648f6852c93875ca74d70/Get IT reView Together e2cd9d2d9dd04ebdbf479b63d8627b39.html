Get IT reView Together

개요

전자 기기를 사려고 리뷰를 찾아보려고 한 결과 리뷰 사이트가 없었다.

블로그 글을 찾아본 결과 검색어로 나오는 대부분이 기업 후원을 받은 글이 대다수였다.

누구나 자유롭게 쓸 수 있는 리뷰 게시판을 만들어 보고 싶다고 생각했다

팀원 및 역활

  • 팀 대표 : 최성현
    • 로그인 화면 구현, 품목별 보드 , 회사별 품목 뷰 기능 구현
  • 팀 원 : 이찬우
    • 로그인 기능 구현, 제품 세부사항 보드, 리뷰 업로드, 리뷰 뷰 기능 구현
  • 팀 원 : 민세미
    • 자유 게시판 기능 구현, 댓글 기능 구현

사용기술

  • Java
  • Apache Tomcat v9.0
  • Spring Framework
  • Mybatis - Oracle
  • bootstrap
  • summernote 텍스트 에디터 API
  • HTML5, JavaScript, CSS3, JSP
  • 네이버 아이디로 로그인
