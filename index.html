교육 사항

OpenAPI를 이용한 빅데이터 전처리·시각화

교육 목표 :

  • 빅데이터를 다루기 위한 개발 언어를 학습하고, 빅 데이터 수집 전처리·시각화 실습과 대시보드 구현 프로젝트를 통해 빅데이터 개발·기획 전문가로 성장할 수 있도록 체계적으로 교육한다.

학습 목표 :

  • Java 프로그래밍과 OOP언어의 특성을 파악하여 객체 지향 언어 지향적인 프로그램 설계 학습
  • 데이터베이스의 주요 구성을 파악하고 SQL언어를 통한 CRUD처리 방법을 익혀 데이터 처리 구현을 파악
  • Web Client 프로그래밍 기술들인 HTML5, CSS5, JavaScript, Ajax를 학습하고 활용 역량 강화한다.
  • Web Server 프로그래밍 기술들인 Servlet, JSP, Spring, Mybatis 를 학습하여, 게시판 프로그램을 완성할 수 있는 역량을 향상시킨다.
  • 통계 분석 언어인 R을 활용한 빅데이터 분석과 시각화를 학습하고 공공 DB와 SNS를 통한 데이터 수집과 웹 크롤링을 통한 데이터 수집 방법을 학습한다.
